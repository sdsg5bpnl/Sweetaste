# Sweetaste

【網頁排版PK戰】甜點電商


### 使用技術

- Gulp 4
- Bootstrap 4


### 相關資源連結

- 緣起活動：[【網頁排版PK戰】甜點電商](https://www.hexschool.com/2018/09/13/2018-09-13-hexschool-challenge/)
- 設計稿：https://lihi.cc/LAfsC

### npm 指令：
- `npm run clean`：清除 dist 資料夾
- `npm run serve`：進入開發模式，包含 Hot Reload 功能
- `npm run build`：輸出專案，會自動壓縮所有檔案
